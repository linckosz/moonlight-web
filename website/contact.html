Get in touch

MoonlightWeb is free and open source, built by one developer. Pick the channel that fits — public on GitHub for anything about the app, or email for private matters.

GitHub — bugs & ideas (recommended)

The fastest way to get help and the best place for anything about the app. Issues are public, so your question and its answer help the next person too — and there's no spam, since a GitHub account is required.

Report a bug / request a feature Open an issue — the tracker for problems and ideas. Browse the project Source code, releases and documentation.

Email — for anything private

For things that don't belong in a public issue — press, partnerships, security reports or a private question — email works too. Expect a slower reply than on GitHub.

✉ Show email address

Robot-proof: the address is assembled by your browser when you interact with the button, so it never appears in the page's source code. Address-harvesting bots — which don't run scripts and just scrape the raw HTML — never see it, which keeps the inbox spam-free.

Support the project

MoonlightWeb is free, but the shared moonlightweb.top DNS server and domain cost real money to keep online. If the project is useful to you, a coffee helps a lot. ☕
