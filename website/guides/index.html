Install guides

Install MoonlightWeb — pick your platform

MoonlightWeb is a free, open-source (GPL-3.0) server that streams your Sunshine gaming rig to any device with a browser — phone, tablet, laptop, TV. One download on the gaming PC. Nothing to install on any other device — just a URL.

Choose your operating system

Install on WindowsGuided installer — Sunshine included in the wizard. Install on macOSApple Silicon guide with the .pkg installer. Install on Linux.deb, .rpm or portable AppImage.

What you need

  • A gaming PC with a GPU capable of hardware encoding — the machine that runs your games (Windows, macOS Apple Silicon, or Linux).
  • Sunshine as the streaming host. Don't have it yet? The MoonlightWeb installer can set it up for you.
  • Any device with a modern browser as the client: Chrome, Edge, Safari or Firefox. No app, no download, no account.
Not sure which one? Install on the PC that runs your games — that's the only machine that needs the MoonlightWeb server. Every other device just opens a browser.
