How it works

Run the server where Sunshine runs, open a browser, pair, and stream.

Run the server

On the machine where Sunshine runs. A tray icon appears.

Open a browser

At https://localhost, your LAN IP, or your domain.

Pick a host

Hosts are discovered via mDNS — or add an IP manually.

Pair & stream

Enter the PIN, choose an app, and play.

The C++/Qt backend embeds moonlight‑common‑c: it speaks GameStream (RTSP/RTP/ENet) to Sunshine and relays video/audio/input to the browser over WebRTC, with a WSS fallback. Video decodes in WebCodecs + WebGPU/canvas, audio in AudioWorklet.
