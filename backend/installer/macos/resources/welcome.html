MoonlightWeb

This installer sets up MoonlightWeb — the browser-based Sunshine/GameStream client — and, if you wish, the Sunshine streaming server it connects to.

You will be asked for Sunshine's credentials and whether to publish a secure public link. MoonlightWeb then pairs with Sunshine automatically the first time it launches.

Cet installeur configure MoonlightWeb (client Sunshine/GameStream dans le navigateur) et, si vous le souhaitez, le serveur de streaming Sunshine. L'appairage se fait automatiquement au premier lancement.
